About

Matika is a high-performance, plugin-agnostic framework built on FastAPI and TypeScript. It provides a zero-knowledge core — authentication, RBAC, dynamic navigation, and security — that can be extended at runtime by AppLugs (plugins) without the core having any prior knowledge of them.

▸ Dynamic plugin discovery & loading
▸ Role-Based Access Control (RBAC)
▸ Menu Hub with selector & role caching
▸ CSRF, rate limiting & security headers
▸ SQLite → PostgreSQL via one env var
▸ Alembic versioned schema migrations
▸ Internationalization with plugin overrides
▸ bcrypt auth · JWT · OAuth (Google / GitHub)
